bbGrid v0.1b

That's an extendable grid system (jqGrid like) developed on Backbone.js, Twitter Bootstrap and jQuery frameworks.

Quick Start

Requirements

  • Backbone.js v0.9.2
  • Twitter Bootstrap v2.2.1
  • jQuery v1.8.3

Usage

Include bbGrid.js and bbGrid.css into your project. Use it!
